IEEE 1588 Products & Implementations

This page provides pointers to web sites describing IEEE-1588 (PTP) products or other material useful in implementing the standard. Implementers of IEEE 1588 are encouraged to supply appropriate material for inclusion on this page. Please provide a pointer to a web site along with a short description of the site contents. Please send this material to Kang Lee (link sends e-mail). 

The following are pointers to web sites for commercial products described as implementing IEEE 1588. NIST does not necessarily endorse the views expressed or the facts presented on this site. Further, NIST does not endorse any commercial products that may be advertised or available on this site. The sites are listed in alphabetical order by company or institution.

ARG Electrodesign Limited (link is external): 1Gbs and 10Gbs Carrier Class IEEE1588v2 switches with optional GPS and SyncE

Bartky Networks (link is external): Software protocol stack and consulting for IEEE 1588 and IEEE 802.1AS. Specializing in PTP on Freescale MPC8313E chip and PTP for Audio/Video Bridging usage.

Bustec (link is external): A complete line of LXI data acquisition and test products supporting IEEE-1588 with the ProDAQ 6100 LXI carrier module.  

Calnex Solutions: providing comprehensive IEEE 1588v2 testing and emulation solutions at line rates from 100Mb/s to 100Gb/s. The Calnex Paragon-X (link is external) supports both the IEEE1588v2 "end-to-end" profile as well as "peer-to-peer" profiles. The transportable Calnex Sentinel (link is external) is used for IEEE 1588v2 field maintenance and troubleshooting.

Conemtech (link is external): Packet Network Sync Module M50-34 - Multi Time Source Time and Frequency Controller.

Domain Time (link is external)  Complete IEEE 1588 software implementation and monitoring for Windows, scalable for the Enterprise.

EndRun Technologies -- provides high performance IEEE 1588 PTP Grandmaster solutions on the Sonoma Network Time Server (link is external) and Meridian II Precision TimeBase (link is external) product lines referenced to GPS or CDMA.

FEI-Zyfer, (link is external) Inc. offers two IEEE 1588-2008 PTP v2 compatible product lines, the PTPSync III GPS referenced (SAASM or C/A) 1588 system and an IEEE-1588 option module for the CommSync II family of systems.

Hilscher (link is external): The network controller netX containing an IEEE 1588 hardware support block.

Intel (link is external): Processors - EP80579 Intel-Architecture-based System on Chip , IXP465 microprocessor , Ethernet Controllers - 1G: 82574, 82576, 82580  10G: 82599

Ixia Anue 3500: Real-world testing solutions for validating IEEE 1588 / PTPv2

IXXAT (link is external): A software protocol stack implementing IEEE 1588.

Korusys (link is external) Ltd: Ltd has released a PTP PCI card which implements telecoms grade clock recovery as a PCI plug-in card or stand-alone OEM module.

Masterclock, Inc. (link is external) : The GMR1000 IEEE 1588 PTP Grandmaster Clock/NTP Server (link is external) and the GMR5000 IEEE 1588 PTP Grandmaster Clock/NTP Server (link is external) with optional GPS/GLONASS receiver can also provide NMEA 0183, IRIG-B, PPO, and 10 MHz outputs.

Meinberg: Combined IEEE1588 Grandmaster Clock/NTP Time Server with GPS (link is external), Combined IEEE1588 Ordinary Clock/NTP Time Server (link is external), Product starter kit (link is external), and PTP client SyncBox (link is external)

Microsemi (link is external): IEEE 1588 Grandmaster Clock supports PTP version 2 and delivers nanosecond time and frequency synchronization across standard Ethernet Networks. Microsemi acquired Symmetricom.

Moxa (link is external): Full line of IEEE 1588 (software and Hardware) enabled industrial managed Ethernet switches (Boundary Clock and Transparent Clock).

Napatech Onboard IEEE 1588-2008 Support (link is external): Napatech's NT20E2-PTP has onboard IEEE 1588-2008 v2 Precision Time Protocol (PTP) support that can be used for applications requiring nanosecond time stamping and time synchronization.

National Instruments: A PCI NIC card with IEEE 1588 (link is external) , A PXI card supporting IEEE 1588, GPS (link is external), and IRIG

OnTime Network (link is external): The OnTime IEEE 1588 Std 2008 (PTP) protocol stack for Grand Master-, Transparent-, Ordinary- and Slave only Clock operation.

PTPd (link is external): Source code for a software implementation of IEEE 1588.

Real-Time Systems GmbH (link is external): IEEE 1588 Precision Time Protocol (PTP) Master Implementations.

Semtech (link is external): Three ToPSyncTM semiconductor products: ACS9510, ACS9593 and ACS9550

Texas Instruments (link is external): The DP83640 Precision PHYTER Ethernet PHY chip with embedded IEEE 1588 hardware and application support.

Time and Frequency Solutions: (link is external) PTP80 Grandmaster clocks

Wireshark: (link is external) A network protocol analyzer available for download that supports IEEE 1588.
